```# COA Tutor 📚

**COA Tutor** is a lightweight, custom-built web application designed to track syllabus progress, specifically tailored for Computer Organization and Architecture (COA). Built using pure frontend technologies, it provides an intuitive way to visualize what topics have been mastered and what is left to conquer.

---

## 🚀 Features

* **Syllabus Tracking:** Clear checklist of COA modules and topics.
* **Progress Visualization:** Instantly see completed vs. remaining topics.
* **Persistent Storage (Optional/Planned):** Keeps track of your progress even after refreshing the browser using `localStorage`.
* **Clean UI:** Simple, distraction-free design built for focused studying.

---

## 🛠️ Tech Stack

* **HTML5:** Structures the syllabus modules and tracking components.
* **CSS3:** Provides a clean, modern, and responsive layout.
* **JavaScript (ES6+):** Handles the interactive checklist logic and progress calculation.
